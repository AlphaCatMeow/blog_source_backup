- [x] 三角形的高、中线与角平分线、三角形的稳定性；锐角三角形的三条高线交于一点，三条高线都在三角形的内部；直角三角形的高交于直角顶点；钝角三角形的高不相交于一点，它们所在的直线相交于一点。
- [x] 三角形的内角；三角形内角和是180度；三角形的外角
- [x] 多边形及其内角和；正多边形内角和都相等，正多边形的每条边都相等
- [x] 全等三角形：能够完全重合的两个三角形叫做全等三角形
